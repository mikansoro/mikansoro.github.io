Penlight Color Values for Liella! and Aqours (King Blade x10V)

2024-04-13
#Liella!#Love Live!#Aqours

I figured I'd post the values I use to mimic Liella! and Aqours penlights using the KingBlade X10v Bluetooth programmable penlight.

All comparisons and testing were done using fully charged, standard eneloop AAA batteries.

I plan to either extend this post or make a follow-up including all missing colors from the Unit Koshien penlight at some point (Nijigasaki, Hasunosora, Saint Snow, Sunny Passion, and subunit colors).

If you're looking for color values to use in a graphic design project, check out Jonbaruby's blog post. This post by github user ratius also has a historical reference of color values used by Love Live staff in the past (Japanese only).
